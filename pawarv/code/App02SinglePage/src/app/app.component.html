Message from app.component.html :

What you see below the solid line comes from router-outlet.

Imagine that the router-outlet is replaced by a component.

Which component? It depends on which link you click.

If you click on Home, i.e the link 'localhost:4200', then the router-outlet will be replaced by app-home which results in home.component.html being rendered with help from HomeComponent.

If you click on About, i.e the link 'localhost:4200/about', then the router-outlet will be replaced by app-about which results in about.component.html being rendered with help from AboutComponent.
